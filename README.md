# Aramisauto — Prototype page d'accueil

Prototype d'accueil Aramisauto avec un hero personnalisé à 3 onglets (Récemment vues · Vous avez recherché · Recommandées pour vous), aux couleurs officielles de la marque.

## Voir la démo

Une fois GitHub Pages activé : `[https://<ton-user>.github.io/<nom-du-repo>/](https://cpasbanal.github.io/aramisauto-proto-hp/)`

## Contenu

- `index.html` — le prototype, autonome (HTML/CSS/JS dans un seul fichier).

## Publier sur GitHub Pages

1. Crée un repo (public) sur GitHub.
2. Ajoute `index.html` à la racine.
3. Repo → **Settings → Pages** → *Branch* : `main` / `/ (root)` → **Save**.
4. L'URL publique apparaît après ~1 min.
